
# OptiRFP Troubleshooting Guide

This guide provides solutions to common issues you might encounter while using OptiRFP.

## Login and Account Issues

### Unable to Login

**Symptom**: Login attempts fail repeatedly.

**Solutions**:
1. Verify you're using the correct email address
2. Reset your password using the "Forgot Password" link
3. Check if your account has been verified (check your email)
4. Clear browser cookies and cache, then try again
5. Try a different browser or device

### Account Locked

**Symptom**: Message indicating your account is locked.

**Solutions**:
1. Wait 30 minutes and try again
2. Check your email for account recovery instructions
3. Contact support if the issue persists

## Document Upload Issues

### Upload Failures

**Symptom**: RFP documents fail to upload.

**Solutions**:
1. Check file size (must be under 50MB)
2. Verify file format (PDF, DOC, DOCX, TXT supported)
3. Try converting your document to a different supported format
4. Check your internet connection
5. Try a smaller document to test if the issue is file-specific

### Analysis Never Completes

**Symptom**: Document uploads but analysis remains "In Progress" for more than 5 minutes.

**Solutions**:
1. Refresh the page to check if analysis has completed
2. Check if the document is password-protected (remove protection)
3. Try with a smaller or simpler document
4. Contact support if the issue persists with multiple documents

## AI Generation Issues

### Content Generation Fails

**Symptom**: Clicking "Generate" doesn't produce any content.

**Solutions**:
1. Verify your subscription hasn't expired
2. Check if you've reached your daily generation limit
3. Refresh the page and try again
4. Ensure your project has a completed RFP analysis
5. Try with shorter/simpler content requirements

### Poor Quality Content

**Symptom**: Generated content is generic or misses key requirements.

**Solutions**:
1. Ensure your RFP document is clearly formatted
2. Add more context in the project details
3. Try regenerating the content
4. Break complex sections into smaller, more focused sections
5. Add specific requirements to the generation prompts

## Subscription and Billing Issues

### Payment Declined

**Symptom**: Unable to upgrade or renew subscription.

**Solutions**:
1. Verify your payment information is correct
2. Check with your bank for any declined transactions
3. Try a different payment method
4. Clear browser cache and try again
5. Contact support if issues persist

### Feature Access Issues

**Symptom**: Unable to access features that should be available on your plan.

**Solutions**:
1. Verify your current subscription in Account Settings
2. Check if your subscription is active and not expired
3. Log out and log back in
4. Clear browser cache
5. Contact support with details of the inaccessible feature

## Performance Issues

### Slow Loading

**Symptom**: Pages or features take a long time to load.

**Solutions**:
1. Check your internet connection
2. Close other browser tabs and applications
3. Clear browser cache and cookies
4. Try a different browser
5. Disable browser extensions that might interfere

### Browser Compatibility

**Symptom**: Features don't work correctly in your browser.

**Solutions**:
1. Update your browser to the latest version
2. Try using Chrome, Firefox, Edge, or Safari
3. Disable browser extensions that might interfere
4. Enable JavaScript if it's disabled
5. Try private/incognito mode to rule out extension issues

## Contacting Support

If you're experiencing persistent issues not resolved by this guide:

1. Email: support@optirfp.com
2. Include:
   - Your account email
   - Detailed description of the issue
   - Steps to reproduce the problem
   - Screenshots if applicable
   - Browser and operating system information

Support hours: Monday-Friday, 9am-5pm EST. Typical response time is within 24 hours (faster for Pro plan subscribers).
